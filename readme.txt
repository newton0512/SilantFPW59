Краткое описание API:
api/cars/ - список всей техники
api/car/{id}/ - информация о технике gj id
api/{user}/cars/ - список техники по клиенту
api/maintenances/ - вся информация по техническому обслуживанию
api/maintenance/{id}/ - информация по техобсуживанию по id
api/{user}/maintenances/ - информация по техобслуживанию техники по клиенту
api/complaints/ - все рекламации
api/complaint/{id}/ - информация по рекламации по id
api/{user}/complaints/ - информация по рекламациям по клиенту
